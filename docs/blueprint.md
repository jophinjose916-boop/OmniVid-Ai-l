# **App Name**: OmniVid AI

## Core Features:

- Multilingual Text-to-Video Creation: Allows users to generate high-quality videos from text prompts, supporting over 100 input languages, leveraging either Stable Video Diffusion or Google Veo API.
- AI-Enhanced Prompt Optimization Tool: Utilizes a built-in LLM (like Gemini) to translate and optimize user prompts into 'AI-friendly' English tokens for improved video generation results.
- Multi-Lingual AI Voiceovers: Provides multi-dialect voice synthesis to generate natural-sounding voiceovers matching the chosen output language of the video.
- Secure Google Authentication: Enables secure one-tap login via Google OAuth 2.0 (Firebase Auth) without storing user passwords on the application servers.
- Content Safety Scanning: Integrates Google Cloud Video Intelligence API to automatically scan and block illegal or unsafe content, including deepfakes and violence.
- Video Playback and Management: Allows users to view, download, share, and manage their generated videos, which are stored securely and encrypted in Google Cloud Storage.
- Tiered Generation System: Offers unlimited free video generation through a hybrid model, including 'Standard Mode' (720p with longer queue) and 'Ad-Supported Queue' (Fast Pass generation after watching an ad).

## Style Guidelines:

- Primary accent color: A vibrant yet deep digital purple (#B257E5), symbolizing AI innovation and creativity.
- Background color: A subtle, desaturated dark plum (#271E2B), providing a sophisticated canvas for video content and interface elements in a dark theme.
- Secondary accent color: A soft, almost pastel blue (#A8A8E6) analogous to the primary, used for interactive elements and highlights to create contrast.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-forward, and impactful feel, ideal for showcasing product features and calls to action.
- Body font: 'Inter' (sans-serif) to ensure excellent readability and a neutral, objective aesthetic for all textual content.
- Sleek, minimalist line-art icons that communicate core functionalities clearly while maintaining a high-tech and sophisticated visual language.
- Spacious, responsive, and intuitive interface with a card-based design for managing prompts and generated videos, ensuring clear visual hierarchy across all devices.
- Smooth, subtle transitions and micro-interactions for state changes, loading indicators, and user feedback, enhancing the modern and responsive feel of the application.
